# Operating Focus (2026-03-11)

## Baseline

- 현재 기본 운영 기준은 `중기`다.
- `단기`, `장기`는 별도 저장 후 참고용으로 유지한다.
- 이유:
  - 최근 백테스트에서 `중기` 타입이 현재 시스템과 가장 잘 맞았다.
  - 평균 수익, 보유 기간, 큰 파동 포착이 가장 균형적이었다.

## Mid-Term Standard

- 매수 해석:
  - 일봉 `60 / 120` 지지 중심
  - 주봉/월봉이 완전히 꺾이지 않은 구조
- 매도 해석:
  - `120일선 이탈`을 메인 매도로 본다
- 실전 기준 보유 기간:
  - 대략 `40 ~ 120 거래일`
  - 최근 백테스트 평균 보유일은 약 `52 ~ 64 거래일`

## Reference Only

- `단기`
  - 급등/가속 전용
  - `5 / 20 / 60` 분할 매도

- `장기`
  - 전환/장기이평 반응형
  - `240 / 480` 이탈 대응

## Practical Use

- 당분간은 `중기`를 메인 규칙으로 보정한다.
- `단기`, `장기`는 케이스가 더 쌓일 때까지 저장만 하고 강하게 최적화하지 않는다.
