# Criteria Snapshot (2026-03-11 Regime Version)

## Core Principle

- 매매의 기본 기틀은 이평선
- 일봉이 메인
- 월봉/주봉은 방향과 확률을 높이거나 깎는 상위 프레임
- 시간봉은 테스트/보조 확인용

## Current State Split

- `강세`
  - 월봉 5선 위
  - 주봉 60선 위
  - 일봉 20선 > 60선
  - 핵심 이평: `20 / 60`

- `중간`
  - 완전 강세는 아님
  - 구조는 살아 있음
  - 핵심 이평: `60 / 120`

- `약세`
  - 월봉 5선 아래이거나
  - 일봉 120선 아래
  - 핵심 이평: `240 / 480`

## Current Weight Direction

- 강한 축
  - 월봉 5선
  - 주봉 60선
  - 52주 신고가 근처
  - 상태에 맞는 핵심 이평선 접촉

- 보조 축
  - 거래량 감소
  - 박스권 정리
  - 고점 대비 눌림

- 강한 감점
  - 52주 신저가 근처
  - 월봉 5선 아래 데롱데롱
  - 머리 위 장기이평 부담
  - 480 아래 구조 붕괴

## Current Timeframe Interpretation

- 월봉: 방향 필터 1순위
- 주봉: 방향 필터 2순위
- 일봉: 메인 진입
- 4시간: 보조 확인
- 1시간: 미세 조정

## Current Notes

- 좋은 자리와 실제 누를 수 있는 자리는 다르다
- 결과가 안 좋았던 유형은 별도 감점이 필요하다
- 빗각은 직접 못 보므로 박스권, 눌림 길이, 거래량, 장기이평 부담으로 근사한다
- 코인은 펀더멘털 점수 제외
- 펀더멘털은 현재 데이터 환경상 참고용 약한 보정만 유지
