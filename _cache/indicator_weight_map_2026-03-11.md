# Indicator Weight Map (2026-03-11)

## Daily Core Indicators

- `dist20`
  - 강세 상태에서 핵심
  - `-4.5% ~ +1.5%` 접촉 가점

- `dist60`
  - 강세/중간 상태 공통 핵심
  - `-2.5% ~ +3.5%` 접촉 가점

- `dist120`
  - 중간 상태 핵심
  - 약세 상태에서는 약한 구조 확인용

- `dist240`
  - 약세/전환 상태 보조
  - 머리 위 부담 감점도 같이 봄

- `dist480`
  - 약세/장기 전환 상태 핵심
  - `480` 아래 구조 붕괴는 강한 감점

- `rsi14`
  - 메인 필터가 아니라 상태 확인용
  - 대체로 `34 ~ 56` 선호

- `vr20`
  - 거래량 감소 확인
  - `1.0 이하` 강한 가점
  - `1.0 ~ 1.2` 약한 가점

- `box_range_pct`
  - 박스권/정리 구간 확인
  - 대체로 `10 ~ 32` 가점

- `from_hh20`
  - 최근 고점 대비 눌림
  - 빗각 대체 흔적 중 하나

- `from_52h`
  - 52주 신고가 근처 여부
  - 약한 가점

- `ret14`
  - 최근 과열 여부 확인
  - 너무 급한 상승은 감점

- `monthly ma20`

## Regime Mapping

- `강세`
  - 월봉 `5선 위`
  - 주봉 `60선 위`
  - 일봉 `20선 > 60선`
  - 핵심 진입선: `20 / 60`

- `중간`
  - 완전 강세는 아니지만 구조 유지
  - 핵심 진입선: `60 / 120`

- `약세`
  - 월봉 `5선 아래` 또는 일봉 `120 아래`
  - 핵심 진입선: `240 / 480`

## Weak / Reference-Only Indicators

- `시장/지수`
  - 차트보다 약한 보조
  - 환경이 아주 나쁘면 약한 감점, 좋으면 약한 가점

- `펀더멘털`
  - 현재 데이터 소스상 장기 백테스트가 어려워 약한 참고용

- `배당`
  - 보조 참고용

- `코인 펀더멘털`
  - 점수 제외
