# 차트 패턴 중간 정리

기준일: 2026-03-11
누적 사례: 127

## 현재 프로젝트 형태

- `v1`은 로컬 전용 별도 시스템이다.
- 지금 다루는 것은 `v3 차트 점수화`이며, 구조는 `티커 1개 입력 -> 현재 자리 평가 -> 과거 사례 비교`다.
- `v3`는 스크리너와 분리된 경량 앱이어야 하며, 조건검색 UI를 다시 붙이지 않는다.
- 현재 운영 URL은 `https://dividend-chart-score.onrender.com/` 이다.
- 차트 점수화 핵심 파일:
  - `chart_score_app.py`
  - `render.yaml`
  - `requirements.txt`
  - `kgw_trend_template_v1.pine` (Pine v6)

## 다음 세션에서 꼭 기억할 것

- 목표는 `모델이 먼저 고른 자리`와 `사용자가 인정하는 자리`의 일치율을 `95% 이상`으로 끌어올리는 것이다.
- 현재 자리 평가는 보조이고, 우선순위는 `과거 정답 자리 매칭`이다.
- 작업 순서:
  1. 종목 선택
  2. 모델이 과거 후보 날짜 제시
  3. 사용자가 `정답 / 근접 / 제외 / 결과 좋음 / 결과 나쁨`으로 보정
  4. 공통 특징 재추출
- 빗각은 직접 수치화하지 못하므로, `고점 대비 위치`, `박스권 길이`, `이평 간격`, `반복 도전`, `거래량`, `ATR` 같은 흔적으로 근사한다.
- 이미 저장된 케이스 날짜를 `새 분석 결과`처럼 재사용하면 안 된다.
- 최근엔 `같은 날짜를 여러 종목에 반복 제시`하는 문제가 있었으므로, 다음 세션부터는 `종목 고유 파동`을 더 강하게 봐야 한다.

### breakout_setup_score 목적

- `공포 뒤 급등 시작 자리`를 따로 점수화한다.
- 메인 매수 자리 점수와 분리해서 검증한다.
- 카카오/위메이드류에서는 반응이 괜찮았고, 에코프로/장기하락형은 아직 추가 보정이 필요하다.
- 매칭률이 높고 결과가 좋으면 채택, 아니면 폐기한다.

## 핵심 결론

- 가장 좋은 눌림은 월봉 강세, 주봉 강세 또는 중립, 일봉만 20일선~60일선으로 눌리는 자리였다.
- 좋은 자리의 공통점은 일봉 60일선 붕괴가 없거나 짧았고, 조정 중 거래량이 줄어드는 경우가 많았다.
- 일봉, 주봉이 동시에 크게 무너진 자리는 추세 눌림보다 낙폭과대 반등 성격이 강했다.
- 자동매매 1차 대상은 `A형 추세 눌림`, `B형 60선 지지 반등`, `C형 이평 수렴 돌파`가 적합하다.

## 유형 분류

### A형: 상위 프레임 강세 + 일봉 눌림

- 월20선 위
- 주20선 위 또는 주60선 위
- 일봉이 20일선 아래로 눌려도 60일선 위 유지
- RSI 35~55 중심
- 조정 중 거래량 감소

예시
- 한국항공우주 `047810` `2025-08-20`
- 현대글로비스 `086280` `2025-11-24`
- 현대로템 `064350` `2025-05-13`

### B형: 20선 이탈 후 60선 지지 반등

- 일봉 20일선은 이탈
- 60일선 또는 120일선 위에서 지지
- 상위 주봉/월봉은 강세 유지
- RSI 25~40도 허용

예시
- 삼천당제약 `000250` `2025-08-21`
- 한미반도체 `042700` `2023-03-15`
- HD현대중공업 `329180` `2024-12-09`

### C형: 이평 수렴 후 재정렬 돌파

- 20일선, 60일선, 때로는 120일선까지 수렴
- 가격이 재안착 또는 돌파
- 큰 추세 시작 전 자주 등장

예시
- 현대로템 `064350` `2025-02-03`
- 한국항공우주 `047810` `2025-02-17`
- 삼성전기 `009150` `2025-06-16`

### D형: 급락 후 반등

- 일봉과 주봉이 같이 흔들림
- RSI 과매도
- 거래량 급증 또는 변동성 확대
- 수익은 날 수 있으나 추세 눌림과는 다른 유형

예시
- KB금융 `105560` `2025-04-09`
- 하나금융지주 `086790` `2025-04-09`
- 현대로템 `064350` `2024-12-10`

### X형: 강도 부족 제외형

- 차트 모양은 A/B/C형처럼 보일 수 있음
- 상위 프레임이 아주 강하지 않음
- 눌림 뒤 반등은 나와도 추세 가속이 약함
- 결과가 짧거나 기대수익 대비 효율이 낮음
- 실전에서는 `매매 제외` 또는 `우선순위 낮음`으로 처리

예시 방향
- SK텔레콤 `017670` 다수 구간

판정 메모
- "도전은 가능하지만 굳이 이 종목을 해야 할 이유가 약한 자리"
- "형태는 맞는데 강도가 부족한 종목"
- "추세주 눌림이 아니라 완만한 박스/저속 우상향"

## 추가 보정 포인트

- `앞의 봉이 너무 없음`은 감점 요소다.
- 즉, 충분한 `박스권/횡보/정리 구간` 없이 바로 솟은 자리의 첫 눌림은 제외 가능성이 높다.
- 같은 이평 지지라도 `박스권 이후 분출`이 더 우선순위가 높다.
- 너무 오른 뒤의 예쁜 차트는 제외한다.
- `좋은 자리`와 `실제로 사기 편한 자리`를 더 분리해야 한다.
- `도전 가능 자리`와 `추천 가능한 자리`를 분리해야 한다.
- `안 샀다`를 곧바로 차트 감점으로 보면 안 된다.
  - `포트 비중`, `종목 수`, `다른 종목 우선` 사유를 분리해야 한다.
- `주봉/월봉이 받쳐줘도 일봉 지지라인이 없으면` 실전 매수는 애매할 수 있다.
- `장기 우하향`, `신저가 진행`, `상승 구조 부재`는 강한 제외다.
- `240일선`은 자주 심리 기준이 되고, 경우에 따라 `480일선`도 본다.

## 결과 라벨

- 앞으로는 자리 평가와 별도로 `결과 좋음 / 결과 보통 / 결과 안좋음`을 같이 본다.
- 목적은 `모양은 맞았는데 왜 결과가 달랐는지`를 수치로 분리하기 위함이다.

### 결과 비교 때 같이 볼 지표

- 진입 시점 `20/60/120 이평과의 거리`
- `5일선 유지 여부`
- `거래량비율(20일 평균 대비)`
- `ATR` 및 변동성 축소 여부
- `20-60 이평 간격`
- `박스권 길이` 또는 정리 구간 유무
- `상승 초기 / 중기 / 후반` 위치
- 시장 강도
- 섹터 강도

### 현재 가설

- 결과가 좋았던 자리는 `박스권 정리`, `이평 상승`, `거래량 감소 후 재증가`, `너무 늦지 않은 위치`가 겹친다.
- 결과가 안 좋았던 자리는 `모양은 맞지만 너무 늦은 위치`, `앞단 정리 부족`, `추세 피로`, `시장/섹터 약화`가 많을 가능성이 있다.

## 점수화 방법 후보

### 1. 규칙 점수

- 가장 단순한 방식
- 각 조건을 충족하면 점수를 더하고, 위배하면 감점
- 장점: 해석이 쉽고 수정이 빠름
- 단점: 조건 간 상호작용을 놓치기 쉬움

예시
- 이평 상승: +15
- 20일선 지지: +20
- 60일선 지지: +15
- 박스권 정리: +15
- 위협구 후 반등: +10
- 거래량 축소 후 회복: +10
- 5일선 분출 유지: +10
- 시장/섹터 강세: +5~10
- 너무 늦은 위치: -20
- 박스권 부족: -15

### 2. 프로토타입 유사도 점수

- 정답 사례들과 현재 자리를 벡터로 비교
- `정답 평균`에 얼마나 가까운지 본다
- 장점: 네가 좋아하는 자리의 결을 반영하기 쉬움
- 단점: 데이터가 적을 때 흔들릴 수 있음

예시 변수
- 5/20/60/120 이격도
- 20/60 이평 기울기
- 20/60 간격
- RSI
- 거래량비율
- ATR 비율
- 박스권 길이
- 최근 20일 최고가 대비 위치

### 3. 결과 확률 점수

- 과거 사례를 `결과 좋음 / 보통 / 나쁨`으로 나누고 분류기처럼 본다
- 장점: 결국 중요한 건 결과라서 방향성이 분명함
- 단점: 라벨링이 필요함

추천 라벨
- 결과 좋음: 20거래일 내 목표 수익 도달, 손절 없이 상승
- 결과 보통: 상승은 했지만 효율 낮음
- 결과 나쁨: 손절 우선 또는 기대수익 미달

### 4. 단계별 점수

- 한 번에 점수 하나를 만들지 않고 단계로 나눔
- 실전적으로 가장 유용

추천 구조
- 구조 점수: 추세, 이평, 박스권, 위협구
- 타점 점수: 20/60/120 지지, 5일선 유지, 거래량
- 환경 점수: 시장, 섹터, 종목 강도
- 결과 점수: 이후 10일/20일 성과

## 인터넷 자료와 비교해 채택할 부분

- Lo, Mamaysky, Wang (2000): 패턴 인식을 시각 판단이 아니라 체계적 특징 추출로 바꾸는 접근을 채택
- Brock, Lakonishok, LeBaron (1992): 이동평균/가격 돌파 같은 단순 규칙도 정보가 있음을 뒷받침
- Moskowitz, Ooi, Pedersen (2012): 추세 자체를 독립 요인으로 보는 방향 채택
- Lee, Swaminathan (2000): 거래량을 같이 봐야 모멘텀 해석력이 높아지는 방향 채택
- CMT JOTA 2020 Ranked Model: 모멘텀, 변동성, ATR 트렌드/브레이크아웃을 `랭킹 모델`로 결합하는 방식 채택

## 현재 추천안

- 최종 한 개 점수보다 `다중 점수`가 낫다.
- 추천 출력 형식:
  - 구조 점수 0~100
  - 타점 점수 0~100
  - 환경 점수 0~100
  - 결과 기대 점수 0~100
  - 급등 전조 점수 0~100
  - 총점 0~100

### 총점 예시 가중치

- 구조 35
- 타점 30
- 환경 15
- 결과 기대 20

### 왜 이 방식이 맞는가

- 네 기준은 단순한 RSI/이평 필터가 아니라 `자리의 질`을 본다.
- 따라서 단일 점수보다 `구조`, `타점`, `환경`, `결과`를 분리해야 설명력이 높다.

## 룰 초안

### 매수 후보

- 월봉 종가 > 월20선
- 주봉 종가 > 주20선 또는 주60선
- 일봉 종가 > 60일선
- 일봉은 20일선 부근 또는 20일선 아래 -5% 내외 조정
- RSI(14) 30~55
- 조정 중 거래량 <= 20일 평균
- 반등 양봉 고가 돌파 또는 20일선 종가 회복 시 진입

### 제외 조건

- 일봉 종가 < 60일선이 2일 이상 지속
- 주봉 종가 < 주20선이며 주봉 RSI 약세
- 하락 거래량이 평균보다 크게 증가
- 20일선과 60일선이 동시에 하락 기울기

## 메모

- 빗각은 해석 요소가 커서 HTS 조건검색의 1차 필터보다는 수동 확인 항목으로 두는 것이 현실적이다.
- 자동 필터는 이평, 거래량, RSI, 가격 위치 중심으로 두고, 지지저항과 추세선은 2차 검토로 넘기는 구성이 적합하다.

## 실사용 메모

- 현재는 `완전자동 매수 시스템`이 아니라 `후보군 압축 + 현재 자리 점수화` 도구로 쓰는 단계다.
- Render 배포 후에는 차트 점수화 화면에서:
  - 티커 입력
  - 현재 자리 점수 확인
- 코멘트는 최근 수정되어 아래를 짧게 설명한다.
  - 이평이 아래서 받쳐주는지, 위에서 막는지
  - 편한 자리인지, 애매한 자리인지
  - 급등 전조 흔적이 있는지
  - 120/240선이 심리적으로 어떤지

## 키움 HTS 조건검색 적용성

- 적용 가능: 일봉, 주봉 기반 이동평균 조건, 가격-이평 비교, 이평 추세, 거래량 이평 추세, RSI와 이격도 같은 보조지표, 괄호와 OR를 통한 조합
- 부분 적용: 월봉 강세 + 주봉 강세 + 일봉 눌림 같은 다중 시간프레임 조합
- 적용 어려움: 추세선 각도, 수평 지지저항 해석, 눌림목의 질 같은 정성 판단

### 실무 적용 방식

- 1차 조건검색: 일봉/주봉 이평, 거래량, RSI, 이격도
- 2차 눈검토: 월봉 위치, 추세선, 수평 지지저항
- 3차 검증: 성과검증, 반복검증, 계좌 시뮬레이션

### 키움용 1차 검색식 방향

- A형: 주봉 20주선 상승, 주봉 종가 >= 20주선, 일봉 종가 >= 60일선, 일봉 종가가 20일선 대비 -5%~+2%
- B형: 일봉 종가 >= 60일선, 일봉 RSI 25~40, 주봉 종가 >= 20주선 또는 60주선
- C형: 일봉 20일선과 60일선 이격 축소, 종가가 두 이평 위로 재안착

### 키움에서 바로 쓰기 어려운 항목

- 월봉까지 포함한 완전한 3중 필터를 한 식에 강하게 묶는 방식
- 추세선 지지, 저항선 박스, 스윙고점/저점 판정
- "눌림이 예쁘다" 같은 정성 판단

## 확률 점수 초안

- 목적: 티커를 넣으면 현재 자리가 A형, B형, C형에 얼마나 가까운지 0~100으로 본다.
- 성격: 자동 판정이 아니라 `현재 위치의 적합도 점수`다.
- 해석: `70 이상`은 유의미한 후보, `55~69`는 보조 검토, `54 이하`는 우선순위 낮음

### A형 점수 기준

- 월봉 종가 > 월20선
- 주봉 종가 > 주20선 또는 주60선
- 일봉 종가 > 60일선
- 일봉 20일 이격도 대략 95~102
- RSI 대략 35~55
- 거래량비율 1.2 이하 선호

### B형 점수 기준

- 월봉 종가 > 월20선
- 주봉 종가 > 주20선 또는 주60선
- 일봉 종가 > 60일선
- 일봉 종가 < 20일선
- 일봉 20일 이격도 대략 90~97
- RSI 대략 25~40

### C형 점수 기준

- 월봉 종가 > 월20선
- 주봉 종가 > 주20선 또는 주60선
- 일봉 종가 > 20일선, 60일선
- 20일선과 60일선 간격이 과도하지 않음
- 일봉 20일 이격도 대략 98~103
- RSI 대략 40~60

### 운용 원칙

- A/B/C 점수는 동시에 높게 나올 수 있다.
- 현재 기본 방향은 `가장 높은 점수의 유형`을 1순위로 해석한다.
- D형 급락반등은 자동매매 1차 대상이 아니라서 일단 점수화 대상에서 제외한다.
- X형 강도 부족 제외형은 점수와 별도로 수동 배제 태그로 관리한다.
