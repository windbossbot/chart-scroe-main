# Mid-Term Recalibration (2026-03-11)

## Current Main Track

- 메인은 `중기`
- 기준선은 `buyable >= 65`
- 기본 매도는 `120일선 이탈`
- 기본 보유 범위는 `40 ~ 120 거래일`

## Why

- 타입별 백테스트에서 `중기`가 가장 균형이 좋았다
- 평균 수익, 보유 기간, 큰 파동 포착이 모두 무난했다
- 사용자 현재 운용 기준도 `120일선 이탈 매도`와 가장 잘 맞는다

## App Changes

- 총점 해석을 `운용 점수`로 변경
- 타입별 기준선을 반영
  - `단기 75`
  - `중기 65`
  - `장기 55`
- 화면에 아래 정보를 직접 표시
  - 매매 타입
  - 운용 우선순위
  - 기준선
  - 기본 매도
  - 대략 보유 기간

## Notes

- `단기`, `장기`는 저장만 하고 당분간 메인 최적화 대상은 아니다
- 중기 기준선은 추후 사례 보정에 따라 `65~75` 사이에서 다시 미세 조정할 수 있다
