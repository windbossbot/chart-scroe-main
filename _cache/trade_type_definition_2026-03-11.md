# Trade Type Definition (2026-03-11)

## 목적

- 같은 `좋은 자리`라도 실제 운용은 다르게 해야 하므로 `단기 / 중기 / 장기`로 분리한다.
- `손실 = 매도`로 본다.
- 매수 후 평가도 단순 기간보다 `타입별 매도 규칙`을 같이 본다.

## 1. 단기

- 의미
  - 급등/가속 종목
  - `사자마자 잘 가는지`가 중요
- 기본 매수 해석
  - 월봉 `5선 위`
  - 주봉 `20/60 위`
  - 일봉 `20 > 60`
  - 최근 `30일 상승률`이 살아 있고 `breakout_setup_score`도 높음
- 기본 매도 해석
  - `5일선` 첫 이탈: 1차 일부 매도
  - `20일선` 이탈: 2차 일부 매도
  - `60일선` 이탈: 나머지 정리
- 백테스트용 단순화
  - `5일선 이탈 40%`
  - `20일선 이탈 30%`
  - `60일선 이탈 30%`
  - 최대 보유 `60거래일`

## 2. 중기

- 의미
  - 가장 일반적인 추세 눌림형
  - 현재 시스템의 메인
- 기본 매수 해석
  - 일봉 `60/120` 지지
  - 주봉/월봉이 완전히 꺾이지 않음
- 기본 매도 해석
  - `120일선 이탈`을 메인 매도로 봄
- 백테스트용 단순화
  - `120일선` 종가 이탈 시 전량 청산
  - 최대 보유 `120거래일`

## 3. 장기

- 의미
  - 장기 하락 후 전환, 횡보 후 재상승, 장기이평 반응형
- 기본 매수 해석
  - `240/480` 근처
  - `120 아래`이거나 월봉이 아직 애매함
- 기본 매도 해석
  - `120`은 흔들림 구간일 수 있음
  - `240/480` 이탈을 더 중요하게 봄
- 백테스트용 단순화
  - 진입 시 `240 위`면 `240` 이탈 청산
  - 아니면 `480` 이탈 청산
  - 최대 보유 `240거래일`

## 공통 메모

- `좋은 자리`와 `실제 매수 자리`는 점수와 성격에 따라 달라진다.
- 지금 분류의 목적은 심리적 흔들림을 줄이기 위한 운용 분리다.
- 이후에는 종목별로 `급등형 / 눌림형 / 전환형`을 더 세분화할 수 있다.
